# Algorithm & Data Structure Visualizer - Project Documentation

## Project Overview

**Project Title:** Interactive Algorithm & Data Structure Visualizer  
**Developer:** Yash Karki  
**Technology Stack:** HTML5, CSS3, Vanilla JavaScript (ES6+)  
**Project Type:** Educational Web Application  
**Completion Status:** Fully Functional & Production Ready

## Executive Summary

This project is a comprehensive web-based educational platform designed to help students and developers understand algorithms and data structures through interactive visual animations. The application features three main visualizers covering sorting algorithms, searching algorithms, and fundamental data structures with real-time complexity analysis and step-by-step explanations.

## Key Features & Functionality

### 1. **Sorting Algorithm Visualizer**
- **Algorithms Implemented:** 6 sorting algorithms
  - Bubble Sort
  - Selection Sort  
  - Insertion Sort
  - Merge Sort
  - Quick Sort
  - Heap Sort
- **Interactive Controls:**
  - Adjustable animation speed (5 levels)
  - Custom array input with validation
  - Pause/Resume functionality
  - Real-time statistics (comparisons & swaps)
- **Educational Features:**
  - Time & space complexity display
  - Algorithm explanations
  - Visual step-by-step process

### 2. **Searching Algorithm Visualizer**
- **Algorithms Implemented:** 4 search algorithms
  - Linear Search
  - Binary Search
  - Jump Search
  - Exponential Search
- **Smart Features:**
  - Automatic array sorting for binary search
  - Visual range highlighting
  - Performance comparison metrics
- **User Controls:**
  - Custom target value input
  - Array size adjustment
  - Speed control

### 3. **Data Structure Visualizer**
- **Data Structures Supported:** 6 fundamental structures
  - Array
  - Stack (LIFO operations)
  - Queue (FIFO operations)
  - Linked List
  - Binary Tree
  - Binary Search Tree (BST)
- **Operations Available:**
  - Insert/Delete elements
  - Search functionality
  - Traversal algorithms
  - Memory representation view
- **Visual Elements:**
  - Animated operations
  - Code examples
  - Complexity analysis

## Technical Architecture

### **Frontend Technologies**
- **HTML5:** Semantic structure and accessibility
- **CSS3:** Modern styling with glassmorphism effects
- **JavaScript ES6+:** Modular class-based architecture
- **No External Dependencies:** Pure vanilla implementation

### **Design Patterns**
- **Modular Architecture:** Separate files for each visualizer

### **System Requirements**
- **Browser:** Modern web browser (Chrome 70+, Firefox 65+, Safari 12+)

### **Installation Instructions**

2. Open `index.html` or `landing.html` in web browser

3. **Execute:** Run algorithms and observe visual animations
